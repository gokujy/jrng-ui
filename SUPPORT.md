# Support

## Usage questions

Use [GitHub Discussions](https://github.com/gokujy/jrng-ui/discussions) for installation, component usage, theming, accessibility, SSR, and architecture questions.

## Bugs and feature requests

- Use the bug template for reproducible defects.
- Use the feature template for generic, reusable proposals.
- Search existing issues before creating a new report.

## Security

Follow [SECURITY.md](SECURITY.md) and report vulnerabilities privately.

## Support boundaries

Community support does not guarantee response times or custom application implementation. Reproductions must use public APIs and contain no private data.
