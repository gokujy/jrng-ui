# Public Documentation Website

The JRNG UI documentation website is a generic public site for package overview, documentation entry points, component categories, theme guidance, and optional support.

## Routes

- `/` - Home page overview
- `/docs` - Documentation sections
- `/components` - Component categories
- `/themes` - Theme system overview
- `/support` - Optional support page

## Navigation

The top navigation includes Home, Docs, Components, Themes, GitHub, and Support. The GitHub link is an external placeholder.

## Home Page

The home page includes a hero, install command, usage examples, feature highlights, preview/demo section, optional support section, and footer links.

## Documentation Pages

The docs page provides placeholder sections for Getting Started, Installation, Theme Setup, Configuration, Components, Forms, Data, Overlay, Navigation, Layout, Advanced, Accessibility, and API Reference.

## Support

Support is optional. Documentation, components, examples, and usage content remain free to access. The support button opens the Razorpay payment link in a new tab and the website does not collect extra user data for support contributions.

## Generic Content

Examples and page content use generic names only, such as users, products, orders, tasks, invoices, and analytics.
