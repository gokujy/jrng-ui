# Public Documentation Website

The JRNG UI documentation website is a generic public site for package overview, documentation entry points, component categories, and theme guidance.

## Routes

- `/` - Home page overview
- `/docs` - Documentation sections
- `/components` - Component categories
- `/themes` - Theme system overview

## Navigation

The top navigation includes Home, Docs, Components, Themes, and GitHub. The GitHub link is an external placeholder.

## Home Page

The home page includes a hero, install command, usage examples, feature highlights, preview/demo section, and footer links.

## Documentation Pages

The docs page provides placeholder sections for Getting Started, Installation, Theme Setup, Configuration, Components, Forms, Data, Overlay, Navigation, Layout, Advanced, Accessibility, and API Reference.

## Generic Content

Examples and page content use generic names only, such as users, products, orders, tasks, invoices, and analytics.
