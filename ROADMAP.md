# JRNG UI Roadmap

The roadmap communicates direction rather than guaranteed release dates.

## Current priorities

- Complete documentation and tested examples for all public entrypoints.
- Strengthen admin, dashboard, data, forms, and responsive layout workflows.
- Expand accessibility, SSR, zoneless, and compatibility validation.
- Improve package diagnostics, migration guidance, and release automation safeguards.
- Grow the Angular Admin Starter as a separate-repository-ready reference application.

## Under consideration

- Additional data visualization adapters.
- More advanced table editing and server-side workflow examples.
- Design-token tooling and theme authoring guidance.
- Community-maintained integration recipes.

Submit proposals through the feature request template. Inclusion here does not imply a committed release.
